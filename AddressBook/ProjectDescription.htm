Web Based Contacts (Address Book)
Technologies ASP.NET(C#), MS-SQL Server
Project Size Small
Description

This is a small web application for managing contact details. It is an ideal learning project for beginners in Asp.Net and a good Start up project who wants to know about tier based application.

Project Highlight:
  • Tier Based design Approch.
  • Component Usage
  • Stored Procedure for all DB Task.
